# Pancake Landing Page

## Overview
A minimal landing page that displays only the word "pancake" in large, centered text with a clean design.

## Features
- Display the word "pancake" in large, prominent text
- Center the text both horizontally and vertically on the page
- Clean, minimal design with no additional visual elements
- No interactivity or navigation required

## Technical Requirements
- Static frontend-only application
- No backend functionality required
- No data persistence needed
- Responsive design that works on desktop and mobile devices

## Design Requirements
- Large, readable typography for the word "pancake"
- Centered layout (both horizontal and vertical alignment)
- Clean, minimal aesthetic
- No background images, graphics, or decorative elements
- Simple color scheme
